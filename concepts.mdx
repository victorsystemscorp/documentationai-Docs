## Los bloques de construcción

<Columns cols={2}>
  <Card title="Páginas" icon="file-text">
    Cada página es un archivo `.mdx`. El contenido vive en el repositorio y es seguro editarlo.
  </Card>
  <Card title="Navegación" icon="list">
    La estructura de la barra lateral proviene de `documentation.json`.
  </Card>
  <Card title="Componentes" icon="layout-grid">
    Bloques reutilizables como Callouts, Steps, Tabs y Cards mantienen las páginas coherentes.
  </Card>
  <Card title="Referencia de API" icon="code">
    Si la usas, la pestaña de API lee desde `openapi.yaml`.
  </Card>
</Columns>

## Forma de la navegación (ejemplo)

```json
{
  "tab": "Documentation",
  "groups": [
    {
      "group": "Getting Started",
      "pages": [
        { "title": "Introduction", "path": "introduction" }
      ]
    }
  ]
}
```

## Cómo adaptar esta página

<Steps>
  <Step title="Reemplaza los términos" icon="edit">
    Sustituye "Pages" y "Components" por el lenguaje de dominio real de tu producto.
  </Step>
  <Step title="Añade tus conceptos" icon="plus">
    Crea una sección para las ideas que tus usuarios deben aprender primero.
  </Step>
</Steps>
