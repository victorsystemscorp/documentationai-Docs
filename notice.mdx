## Qué es seguro cambiar

Puedes editar el contenido de cualquier página, renombrar secciones y actualizar la estructura de navegación. Si no necesitas la Referencia de API o el Centro de ayuda, también puedes eliminar esas pestañas de `documentation.json`.

<Columns cols="2">
  <Card title="Contenido de las páginas" icon="file-text" horizontal="false">

    Edita cualquier archivo `.mdx` en este repositorio.

  </Card>

  <Card title="Navegación" icon="list" horizontal="false">

    Actualiza las etiquetas, el orden y las pestañas visibles en `documentation.json`.

  </Card>

  <Card title="Referencia de API" icon="code" horizontal="false">

    Sustituye `openapi.yaml` por tu propia especificación o elimina la pestaña de Referencia de API de `documentation.json` si no la necesitas.

  </Card>

  <Card title="Centro de ayuda" icon="help-circle" horizontal="false">

    Reemplaza las preguntas frecuentes, guías y artículos de solución de problemas por tu contenido de soporte real, o elimina la pestaña de Centro de ayuda de `documentation.json` si no forma parte de tu documentación.

  </Card>
</Columns>

## Cambios iniciales recomendados

<Steps>
  <Step title="Renombra las páginas" icon="edit" title-type="p">

    Actualiza los títulos y descripciones de las páginas para que coincidan con tu producto.

  </Step>

  <Step title="Sustituye los enlaces de ejemplo" icon="link" title-type="p">

    Elimina cualquier URL de relleno y agrega las tuyas.

  </Step>

  <Step title="Elimina pestañas y secciones sin uso" icon="settings" title-type="p">

    Abre `documentation.json` y elimina las entradas de pestañas o secciones que no planeas usar, como la Referencia de API o el Centro de ayuda, sin borrar ningún archivo.

  </Step>

  <Step title="Elimina páginas sin uso" icon="trash" title-type="p">

    Borra las páginas `.mdx` que no planeas mantener para que la barra lateral solo muestre contenido relevante.

  </Step>
</Steps>
