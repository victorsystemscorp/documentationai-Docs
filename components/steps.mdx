# Pasos

Los pasos muestran un flujo lineal para configuraciones y tutoriales.

## Ejemplo

<Steps>
  <Step title="Instalar dependencias" icon="download">
    Ejecuta el comando de instalación para agregar los paquetes requeridos a tu proyecto.
  </Step>
  <Step title="Configurar ajustes" icon="settings">
    Crea un archivo de configuración en la raíz de tu proyecto con tus credenciales de API.
  </Step>
  <Step title="Desplegar documentación" icon="rocket">
    Envía tus cambios y realiza el despliegue a producción.
  </Step>
</Steps>
