## Pestañas

Las pestañas agrupan variantes relacionadas como sistema operativo, idioma o plataforma.

## Ejemplo

<Tabs>
  <Tab title="macOS" icon="apple">
    Instala usando Homebrew para obtener la mejor experiencia en macOS.

    ```bash
    brew install documentation-ai
    ```
  </Tab>
  <Tab title="Windows" icon="monitor">
    Descarga y ejecuta el instalador de Windows desde la página de lanzamientos.

    ```bash
    winget install documentation-ai
    ```
  </Tab>
  <Tab title="Linux" icon="terminal">
    Usa tu gestor de paquetes o descarga el binario directamente.

    ```bash
    sudo apt install documentation-ai
    ```
  </Tab>
</Tabs>
